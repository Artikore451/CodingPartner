RECURSIVE FUNCTION IN C

A recursive functions calls back the function in which it is present. The recursive functions call works until the final task completes. The two cases of recursive solution are:

Base case, where the problem is easily solvable without making any calls for same function. It acts as the terminating condition of the recursion. If base is not present then the function will indefinitely call itself.

Recursive case, where the problem divides into simpler sub-parts. The function also calls back itself with the sub-parts and solves the problem.


Recursion also means breaking a large and complex problem into smaller parts and solving it. The complex problem breaks into simpler problem and explicitly includes it. The basic steps of recursive program is :


Step 1 : Specifying the base case to stop the function from calling itself.

Step 2 : Checking whether the current processing value matches the value of the base case. If the                     value matches then the result processes and the value returns.

Step 3 : Breaking the problem into smaller or simpler sub – problems.

Step 4 : Calling the function from the sub- problems. This helps in further solving of the problem.

Step 5 : Combining the results of the sub – problems.

Step 6 : Also returning the result of the entire problem.


Types of Recursion:

Recursion characterizes based on:

whether the function call is direct or indirect (direct or indirect recursion)

any operation is remaining after each recursive call (tail recursion or not).

the structure of the calling pattern (linear or tree recursion).

1) Direct Recursion:

A function is said to have direct recursion if it explicitly calls itself.

2) Indirect Recursion:

A function has indirect recursion if it contains a call to another function and that function ultimately calls it.
